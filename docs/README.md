# Documentacion de Grimorio

Indice de documentacion activa del proyecto.

## Documentos activos

- `../README.md`: vision general y arranque rapido.
- `../CHANGELOG.md`: historial de cambios.
- `../backend/docs/API_STANDARDS.md`: estandares de API backend.
- `../frontend/README.md`: guia del frontend.
- `../mobile/README.md`: guia del modulo movil.

## Politica de documentacion

- Documentacion global: en `docs/` y raiz.
- Documentacion especifica de modulo: dentro del modulo.
- Documentacion historica o de migraciones cerradas: en `docs/archive/`.

## Mantenimiento

Al cerrar una funcionalidad relevante:

1. Actualizar documento tecnico del modulo afectado.
2. Registrar cambio en `CHANGELOG.md`.
3. Si el documento ya no es vigente, moverlo a `docs/archive/` con una nota de contexto.

# Modulo Mobile

Espacio reservado para aplicaciones moviles de Grimorio.

## Objetivo

Permitir crecimiento por apps moviles sin mezclar codigo con backend o frontend web.

## Estructura sugerida

```text
mobile/
|-- android-waitstaff/
|-- android-customer/
`-- ios/
```

## Regla de documentacion

Cuando se inicie una app movil, crear un README propio dentro de su carpeta con:

- stack tecnologico,
- comandos de build/run,
- convenciones de estructura,
- integracion con backend.
